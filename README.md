# StackOverflow

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.8.

## Run locally

Once you cloned the project, run below commands:

`cd StackOverflow`

`npm install` OR `yarn`

`ng serve --open`
